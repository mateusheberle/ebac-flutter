# Markdown

- [Artigo: ](https://www.alura.com.br/artigos/como-trabalhar-com-markdown) Markdown: como trabalhar com essa linguagem de markup?

- formato simples de markup 
    - marcação de texto
- marcar o que é:
    - importante
    - tópico
    - link
    - imagens
- sem usar marcações mais complexas
    - como HTML
